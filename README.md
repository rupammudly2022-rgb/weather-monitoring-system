# Weather Monitoring System

This project is a basic weather monitoring system using different sensors.

The system is designed to measure temperature, humidity, air pressure,
and UV level using electronic sensors and microcontroller.

Sensors used:
DHT11 – temperature and humidity
BMP180 – air pressure
GUVA-S12D – UV sensor

Description:
The sensors are connected to microcontroller and the data can be used
for environmental monitoring and IoT applications.

Status:
Work in progress. The project is being developed for learning
sensor interfacing and embedded systems.
